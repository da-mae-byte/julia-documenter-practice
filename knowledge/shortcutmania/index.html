ショートカットマニア

Windows

Win + 1～0タスクバーに並んだアイコンの1～10個目に対応。11個目以降も対応
Win + v過去にコピーしたものを選択して貼り付け。<br>上限あり(25件)だがピン止めすると消えない。<br>ピン止めした場合、リブートしても消えない。<br>キャプチャや、Officeの図形などにも対応。
Win + rプログラム名を指定して実行<br>calc:電卓<br>cmd:コマンドプロンプト<br>winver:Windowsのバージョン確認
Win + Shift + s領域を選択してスクリーンショット
Win + PrintScreen全画面を撮影し、"ピクチャ\スクリーンショット"に保存
全角入力時にShift + Space半角スペース

エクスプローラー

Ctrl + Shift + n新しいフォルダーを作成
Shiftキー + ファイルを右クリックファイルのフルパスをコピーするメニューが現れる

サクラエディタ

Alt + a選択した行を一番左側のレコードでソート
Alt + m選択した行を重複削除
Alt + ↑／↓矩形選択

VSCode

Ctrl + ↑／↓行を移動
Ctrl + dファイル内の同じ単語を選択。カーソルが複数になるので同時に編集ができる
Ctrl + Alt + ↑／↓複数行選択。カーソルが複数になるので同時に編集ができる（サクラエディタと同じ感じ）
Ctrl + k u文字列を選択し、Ctrlを押したままk、uと入力。選択した文字列をすべて大文字にする
Ctrl + k l選択した文字列をすべて小文字にする

MobaXterm

Ctrl + Shift + uタブを複製
Ctrl + dログアウト

Chrome

Alt + ←戻る
Alt + →進む
Ctrl + dブックマーク登録
Ctrl + h履歴を表示
Ctrl + t新しいタブを開く

Excel

F2セルを編集
Ctrl + b太字にする
Ctrl + tテーブルを追加

Office系

F4直前に設定した書式を設定。大量の図をサイズ変更するときとかに便利。
