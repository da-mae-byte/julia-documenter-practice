gridレイアウト

gridレイアウトにしたdivの中に入れたDOM要素は、columns×rowsのグリッドになる。 本来はヘッダ、メイン、右メニュー、フッタなどのレイアウトのためのものだが、テーブルに使用することも出来る。 ただ、その際は行数の上限が1000行であることに注意。

<div class="grid">
    <div>a</div>
    <div>b</div>
    <div>c</div>
    <div>d</div>
    <div>e</div>
    <div>f</div>
</div>
.grid {
    /* WebKit */
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;

    /* ms */
    display: -ms-grid;
    -ms-grid-columns: (1fr)[3];
    -ms-grid-rows: auto;
}
cnt = 1;
querySelectorAll('.grid').foreach(function(node) {
    node.style["gridRowStart"] = cnt;   // WebKit 省略可
    node.style["msGridRow"] = cnt;      // ms
    cnt ++;
});
<div class="grid">
    <div style="-ms-grid-row:1">a</div>
    <div style="-ms-grid-row:2">b</div>
    <div style="-ms-grid-row:3">c</div>
    <div style="-ms-grid-row:4">d</div>
    <div style="-ms-grid-row:5">e</div>
    <div style="-ms-grid-row:6">f</div>
</div>
